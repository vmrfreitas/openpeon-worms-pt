# Worms Armageddon (PT-BR) — OpenPeon Sound Pack

![worms](https://c.tenor.com/YkdlYcER2vcAAAAd/tenor.gif)

Brazilian Portuguese voice lines from Worms Armageddon for your coding sessions.

## About

The classic Team17 worm voices, dubbed in Portuguese. 33 sound clips mapped across all 7 CESP categories.

## Included Sounds

- **Olá?!** — Hello?!
- **Sim, senhor!** — Yes, sir!
- **Mayday, Mayday!**

### Task Acknowledge
- **Sim, senhor!** — Yes, sir!
- **Pega essa!** — Take this!
- **Atacar!** — Attack!
- **Fogo!** — Fire!

### Task Complete

## Installation

```bash
peon packs install worms-armageddon-ptbr
peon packs use worms-armageddon-ptbr
```
